8. Responsive

Modifier la mise en page d’un site afin que le contenu s’adapte à l’écran quel que soit le terminal utilisé (smartphone, tablette, ordinateur…), inboxes le rend assez simple en utilisant les valeurs media xs-sm-md-lg-xl-pr qui représentent les différentes tailles des écrans existantes selon leur largeur :

  • xs : petites tailles d’écran moins de 480px
  • sm : les petites tailles d’écran entre [480px , 768px[
  • md : les tailles medium d’écran entre [768px , 992px[
  • lg : les tailles grandes d’écran entre [992px , 1200px[
  • xl : les grandes tailles d’écran plus que 1200px
  • pr : à l’impression
